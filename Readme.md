
# increment

  increment strings

## Example

```js
increment('user', '-') // user-1
increment('user-1', '-') // user-2
increment('user-999', '-') // user-1000
```

## Installation

    $ npm install increment

or in the browser...

    $ component install matthewmueller/increment

## API

### increment(str, [ prefix ])

Increment `str` by 1. Give an optional `prefix`.
